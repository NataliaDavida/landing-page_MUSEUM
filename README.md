# Museum — Modern Landing Page

This project is a modern, responsive landing page for a museum, designed to showcase current exhibitions, schedules, and cultural heritage in an engaging and accessible format.

* [Live Demo](https://nataliadavida.github.io/landin-page_museum/)

---

## Features

* **Responsive Design** — Fully optimized for mobile, tablet, and desktop viewports.
* **Modular Styles** — Built using SCSS with a clean and scalable component structure.
* **BEM Methodology** — Strictly follows BEM naming conventions for clean and predictable styles.
* **Pixel Perfect Layout** — Implemented according to professional design standards and precise grid alignments.

---

## Design Reference

The project is implemented based on the provided Figma layout, following design system specifications, precise typography, and standardized grid spacing.
